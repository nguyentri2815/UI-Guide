Form controls

cung cấp điều khiển dạng văn bản điền vào .... hoặc khoảng trống thẻ span thay cho input

Describe

form-control là thẻ thuộc dạng hiển thị inline-block mặc định có border bottom là dashed

Kích thược width của form-control phụ thuộc vào class w-* or min-w-* mà người dùng tự đặt link tùy chính class sizing

Có thể tùy chỉnh hiển thị border hoặc không có thể tùy chỉnh màu màu sắc border ... link tùy chính class border

Example

                                
    <div class="mb-3">
        <span class="form-control w-0"span>
    </div>
    <div class="mb-3">
        <span class="form-control w-25"span>
    </div>
    <div class="mb-3">
        <span class="form-control w-50"span>
    </div>
    <div class="mb-3">
        <span class="form-control w-100"span>
    </div>
