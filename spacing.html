Margin and padding

class được xậy dựng từ 1-5 đơn vị rem tương ứng giá trị .25rem đến 3rem.

Notation

class được xậy dựng từ 1-5 đơn vị rem tương ứng giá trị .25rem đến 3rem.

cấu trúc class [property][sides]-[size]

property:

  • m - cho thiết lập: margin
  • p - cho thiết lập padding

sides: là một trong

  • t - thiết lập class cho margin-top hoặc padding-top
  • b - thiết lập class cho margin-bottom hoặc padding-bottom
  • s - (start) thiết lập class cho margin-left hoặc padding-left in LTR, margin-right or padding-right in RTL
  • e - (end) thiết lập class cho margin-right hoặcpadding-right in LTR, margin-left or padding-left in RTL
  • x - thiết lập class cho cả *-left và *-right
  • y - thiết lập class cho cả *-top vả *-bottom
  • trống - thiết lập class cho a margin hoặc padding cho cả 4 vị trí

size: là một trong

  • 0 - for classes that eliminate the margin or padding by setting it to 0
  • 1 - (by default) for classes that set the margin or padding to $spacer * .25
  • 2 - (by default) for classes that set the margin or padding to $spacer * .5
  • 3 - (by default) for classes that set the margin or padding to $spacer
  • 4 - (by default) for classes that set the margin or padding to $spacer * 1.5
  • 5 - (by default) for classes that set the margin or padding to $spacer * 3
  • auto - for classes that set the margin to auto

Examples

                                
    .mt-0 {
    margin-top: 0 !important;
    }
    
    .ms-1 {
    margin-left: ($spacer * .25) !important;
    }
    
    .px-2 {
    padding-left: ($spacer * .5) !important;
    padding-right: ($spacer * .5) !important;
    }
    
    .p-3 {
    padding: $spacer !important;
    }
